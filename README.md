# Library web application

Used: Spring Core, MVC, JDBC, Hibernate, Maven, SQL(MySql), DI, IOC

User can create new readers, new books, assign books to readers, show free books, search books by name, reassing books and readers.
To use this application use MySql database and create tables according BookMapper and HumanMapper, then create relation one to many (hunan.id) to (books.humanid).
